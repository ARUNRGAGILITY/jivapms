CSV Format

The CSV file should have the following columns:

  • username/email (required): Username or email of existing user
  • role_name/role_id (required): Name or ID of the role to assign
  • organization_id (optional): Organization ID (will use default if not specified)
  • active (optional): 'true' or 'false' (defaults to 'true')

Example:

username/email,role_name/role_id,organization_id,active
john.doe@example.com,Reviewer,123,true
janedoe,Editor,456,true
user@example.com,Site Admin,,true
